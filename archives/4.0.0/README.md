# QuickMath

Mobile-friendly mental math trainer for square roots with spaced repetition.

## Highlights
- Hamburger menu and bottom tabs for Practice, Stats, History, Tips.
- Large answer input for fast typing (mobile-friendly).
- Stats dashboard with cards, charts, filter, grouped attempts, and activity calendar.
- Version switcher and modal version-history cards.
- Google login/signup, Firestore save and retrieval sync, plus local storage.

## Run
Open `index.html` in a modern browser with internet access.
