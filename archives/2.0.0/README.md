# QuickMath

Mental math app focused on fast square-root drills.

## Features
- Minimal black UI with large question area.
- Hamburger menu for navigation.
- Google login/signup via Firebase Auth.
- Practice mode with auto-advance.
- Statistics dashboard (correct/wrong/time/recent attempts).
- Tips & Method page with colourful mental-math guidance.
- Stores attempt stats in localStorage and Firestore.

## Run
Open `index.html` in a modern browser with internet access.
