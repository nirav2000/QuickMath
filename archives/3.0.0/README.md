# QuickMath

Mental math trainer focused on square roots with progression and analytics.

## Highlights
- Hamburger navigation and minimal black UI.
- Google login/signup via Firebase Auth.
- Local + Firestore save, and Firestore retrieval sync on login.
- Spaced repetition (SM-2 style) with adaptive difficulty progression.
- Statistics dashboard with charts/graphs.
- Detailed step-by-step tips page.
- Version switcher + in-app version history page + archived releases.

## Run
Open `index.html` in a modern browser with internet access.
