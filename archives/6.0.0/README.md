# QuickMath

Mobile-friendly mental math trainer with modular topics and selectable spaced repetition.

## Highlights
- Modular topic system (`topics/*.js`) for easy topic additions.
- Settings page with spaced repetition selection:
  - SM-2 (Strict)
  - SuperMemo (Heuristic)
- Square roots and percentages included.
- Version switcher, stats, tips, and Firebase sync.
