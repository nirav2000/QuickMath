# QuickMath

Mobile-friendly mental math trainer for square roots and percentages.

## Highlights
- Bottom tabs + hamburger menu navigation.
- Version selector uses central version metadata and opens selected version in the same tab.
- Topic selector: Square Roots + Percentages.
- Spaced repetition, stats dashboard, grouped attempts, activity calendar.
- Google login/signup with local+Firestore save and Firestore retrieval sync.

## Run
Open `index.html` in a modern browser with internet access.
