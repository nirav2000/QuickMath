# QuickMath

Mental math app for rapid-fire practice.

## Current topic
- Square roots (perfect squares).

## Features
- Minimal black/white interface.
- Full-screen style question display.
- Auto-advance after each answer.
- Tracks question, answer, correctness, and response time.
- Stores data in Firebase Firestore (`attempts`) with local fallback.
- Learning panel for quick technique review.
- Version switcher + archives.

## Run
Open `index.html` in a modern browser.
